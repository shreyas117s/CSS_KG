

Semantic Tags

Semantic Tags                                
● Meaningful: Describe content.
● SEO: Good for search engines.
● Accessibility: Useful for screen 
readers.
● Examples: <header>, <footer>, 
<article>, <section>, <nav>.


Non-Semantic Tags
● Generic: No specific meaning.
● For Styling: Used for layout.
● No SEO: Not SEO-friendly.
● Examples: <div>, <span>, <i>, 
<b>.
